# Student Grade Tracker

## Description

Student Grade Tracker is a Java console application that helps manage student records. It allows users to enter student names and marks, calculate the average marks, determine the highest and lowest marks, and generate a complete report.

## Features

- Add student names
- Enter student marks
- Calculate average marks
- Find highest marks
- Find lowest marks
- Generate student report
- Grade calculation

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner Class

## How to Run

Compile:

```bash
javac StudentGradeTracker.java
```

Run:

```bash
java StudentGradeTracker
```
